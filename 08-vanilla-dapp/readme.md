# VanillaJS dApp

The program that this vanilla dApp uses is on program/posts/src and is only to be used as reference.

## Requirements
- deps installed in npm version 9.6.7 (node 18.17.1)
- live server extention

## How to run

1. Enter in terminal
```
cd app && npm i
```
2. Enter in terminal
```
npm run start
```
3. Activate live server
